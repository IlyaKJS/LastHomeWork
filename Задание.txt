Задание для тех, у кого нет опыта работы с GIT

1) Для начала нужно зарегистрироваться в Гитхаб:
	https://github.com/signup?ref_cta=Sign+up&ref_loc=header+logged+out&ref_page=%2F&source=header-home
2) Создайте новый публичный репозиторий.
	Кнопка New
	Название для репозитория можете взять любое
	Тип укажите публичный
	Create Repository
	Нещабудьте после этого скопировать ссылку созданного репозитория
3) Откройте текущую папку в VSCode, и на вкладке Git иннициализируйте репозиторий.
4) Откройте в VSCode терминал и подключите ваш удалённый репозиторий к вашему созданному репозиторию
	git remote add origin <ссылка>
5) Создайте файл index.html
	1) Создайте базовую струнктуру html.
	2) Создайте в <body> заголовое <h1>, с текстом Счётчик
	3) Добавьте <div>, в котором добавьте <p> с цифрой 0 и <button> с текстом "Нажать!"
6) Сохраните файл. Перейдите во вкладку Git, проиндексируйте файл, а затем зафиксируйте его с любым соответствующим комментарием
7) Отправьте с помощью push этот коммит в удалённый репозиторий. 
	Для этого, на вкладке Git нажмите на троеточие вверху, и выберите команду push
	Проверьте ваш репозиторий на github, что видите появившийся файл
8) Создайте от ветки main новую ветку css
	Для этого, в нижнем левом углу нажать на название текущей ветки (изначально это будет ветка main)
	В появившемся окне выбрать create new branch и указать имя ветки
	(убедитесь, что находитесь в нужной ветке. Название текущей ветки в нижнем левом углу. Если вы не в той ветки, которая нужна, нажмите на назнание текущей ветки и в появившемся окне выберите нужную)
9) Создайте файл style.css и подключите его в <head>, и в этом файле задайте стили для элементов на странице, застилив их на ваше усмотрение.
10) На вкладке Git проиндексируйте все изменения (создание файла style.css и подключение его в index.html) и зафиксируйте это, с соответствующим комментарием
11) От ветки main создайте новую ветку js и переключитесь на неё
12) Создайте файл script.js и подключите его в <head> (напоминаю про defer)
13) В файле script.js добавьте обработчик событий, который при клике считывает цифру из <p> и увеличавает её на 1
14) Проиндексируйте и зафиксируйте изменения.
15) Отправьте все изменения в удалённый репозиторий
16) Переключитесь на ветку main.
	Поменяйте заголовок страницы в <title>, проиндексируйте и зафиксируйте.
	На вкладке Git нажмите на троеточие и выберите в разделе Branch команду Merge.
		Эта команда может объединять несколько веток.
	Таким образом добавьте в ветку main обе ветки, css и js.
		(Возможно появится сообщение, что есть конфликт. Тогда, нажмите кнопку Show Changes, потом Resolve in Merge Editor. Изучите в верхних двух окнах, что у вас есть в вашей ветке, и что идёт из другой ветки. В нижнем из трёх окон сделайте такой вариант, который вас устраивает, который учитывает обе версии. И после этого нажмите кнопку Complete Merge)
	Незабудьте всё зафиксировать и отправить изменения в удалённый репозиторий
17) Пришлите мне ссылку на ваш репозиторий
